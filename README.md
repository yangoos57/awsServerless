## 문제 정의 및 해결 <a name="consideration"></a>

### 키워드 추출 모델 선정

**문제정의**

<!-- - 원하는 도서를 간편하게 찾기 위한 효과적인 방법 고민 -->

- 하나의 도서를 여러 개의 키워드로 정의한 뒤, 검색 단어와 일치하는 키워드가 많은 순으로 도서를 추천하는 방법 고려

- **도서 핵심 내용을 키워드로 정의할 수 있는 성능을 가진 NLP 모델 필요**

**문재해결**

- 키워드 추출 모델로 TF-IDF, Word2Vec, keyBert 비교

- TF-IDF의 경우 준수한 성능을 보였으나, 다른 도서와 구분되는 키워드를 추출하는 데 목적이 있다보니 학습 텍스트 내에 포함된 사람 이름을 키워드로 추출하는 문제 발생

- Word2Vec의 경우 텍스트별 키워드 추출 보다는 단어 간 연관성을 찾는데 특화된 툴이라 판단해 제외

- keyBert의 경우 개별 문서의 키워드 추출에 특화, 기 학습된 모델을 불러와 사용 가능, 파인튜닝 없이도 준수한 추출 성능 검증
- **효율성과 성능을 모두 겸비한 keyBert를 키워드 추출 모델로 선정**

<br/>

### Word2Vec 활용 도서 검색 성능 향상

**문제정의**

- 사용자가 검색한 단어 개수에 따라 검색 성능에 차이 발생
- 하나 또는 두 개의 키워드로 검색할 경우, 수백권의 도서가 검색됨에 따라 우선순위 선정에 어려움 발생

**문제해결**

- word2vec 활용 검색 단어와 연관성이 높은 키워드 20개를 추출해 도서 검색에 활용
- `파이썬` 검색 시, 코사인 유사도가 높은 20개 키워드(`주피터`, `맷플롯립`, `넘파이`, `판다스` 등)를 추출해 도서 검색에 활용
- 사용자 검색 단어와 W2V 추출 키워드에 중요도를 다르게 설정해 도서 우선순위 선정
- 사용자 검색 단어는 W2V 추출 키워드 대비 3배 가중치 부여
- 검색 단어가 `파이썬`인 경우, `파이썬`, `주피터`, `판다스`가 포함된 도서는 점수는 `파이썬` 3점, `주피터` 1점, `판다스` 1점이므로 5점으로 계산
- `주피터`, `맷플롯립`, `넘파이`가 포함된 도서는 3점으로 계산
- **키워드 점수가 높은 순으로 도서를 추천함으로서 도서 검색 성능 향상**

### Extract: 도서관 장서 데이터 확보 및 도서정보 크롤링

- 정보나루 API 활용 도서관 별 장서 데이터 업데이트
- 기존 장서 데이터과 업데이트 된 장서 데이터를 비교해 신규 도서 목록 확보
- 웹 크롤링을 통해 도서소개, 목차, 추천사 등 키워드 추출에 필요한 텍스트 수집

### Transfrom: word2vec 학습 및 keyBert 활용 키워드 추출

- Konlpy 활용, 확보한 텍스트에 대한 형태소 분석 진행
- 형태소 분석 결과 활용 Word2Vec 학습 및 keyBert로 개별 도서 키워드 추출

### Load: 정제 데이터 DB에 저장

- 개별 단계에서 확보한 데이터를 세 종류의 테이블에 저장
- 하나 더 넣어야 하는데 뭘 넣어야하지

### Word2Vec 활용 연관 키워드 추출

- 사용자가 검색한 키워드를 Word2Vec의 Input 검색어로 활용해 연관성이 높은 키워드 20개를 추출
- Ex) `파이썬` ,`자연어` 검색 시 `딥러닝`, `데이터분석`, `파이토치`, `품사`등 연관성이 높은 단어를 추출
- 추출 된 키워드 활용 도서 검색 및 도서 추천순위 계산에 활용

### 도서 목록 생성

- 사용자가 선택한 도서관의 도서 데이터 및 개별 도서의 키워드 확보
- 확보된 도서 목록과 word2vec에서 추출한 20개의 키워드를 비교, 키워드가 하나 이상 포함된 도서 추출
- 추출 도서의 도서 제목, 저자, 청구기호를 도서 DB에서 검색하여 도서 목록 생성

### 도서 추천순위 계산

- 일치하는 키워드가 많은 도서일수록 높은 순위로 추천
- 사용자 검색 단어와 추출한 키워드의 중요도가 다르기 때문에 사용자 검색 키워드는 3점, word2vec 추출 키워드는 1점으로 계산
- `파이썬`, `자연어` 검색 시, `딥러닝`, `데이터분석`, `파이토치` , `품사` 등 연관 키워드 추출
- `파이썬`, `자연어`, `딥러닝` 키워드를 보유한 도서 A는 7점, `딥러닝`, `데이터분석`, `파이토치` 키워드를 보유한 도서 B는 총 3점 부여
- 점수가 더 높은 A 도서가 B 도서 보다 상위에 추천
